## 🎯 仓库使用规范

1.  **权限说明**：运营团队成员可读写，研发团队仅开放只读权限，禁止非授权人员修改；权限申请联系仓库负责人。

2.  **内容边界**：仅允许上传/链接运营相关非代码内容（活动方案、数据报表、话术库等），严禁提交任何业务代码、技术配置、编译文件。

3.  **更新要求**：所有新文档/资源必须先同步到本索引，再上传对应文件或补充链接，确保索引实时有效，避免无效链接。

4.  **维护责任**：由运营团队指定专人（如运营负责人）每周检查索引，清理过期内容、失效链接，每月汇总更新记录。
